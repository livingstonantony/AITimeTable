## Constraint Satisfaction Model

```
DECISION VARIABLES
в”ңв”Җв”Җ teacher_assign[(teacher, slot)] = Boolean вҲҲ {0,1}
в””в”Җв”Җ subject_assign[(subject, slot)] = Boolean вҲҲ {0,1}

CONSTRAINTS
в”ңв”Җв”Җ Exact Hours
в”Ӯ   в”ңв”Җв”Җ ОЈ teacher_assign[(t, s)] = teacher_hours[t] вҲҖ teachers
в”Ӯ   в””в”Җв”Җ ОЈ subject_assign[(s, i)] = subject_hours[s] вҲҖ subjects
в”Ӯ
в”ңв”Җв”Җ One Per Slot
в”Ӯ   в”ңв”Җв”Җ ОЈ teacher_assign[(t, slot)] = 1 вҲҖ slots
в”Ӯ   в””в”Җв”Җ ОЈ subject_assign[(s, slot)] = 1 вҲҖ slots
в”Ӯ
в”ңв”Җв”Җ Daily Limits
в”Ӯ   в”ңв”Җв”Җ ОЈ teacher_assign[(t, daily_slots)] вүӨ max_teacher вҲҖ days
в”Ӯ   в””в”Җв”Җ ОЈ subject_assign[(s, daily_slots)] вүӨ max_subject вҲҖ days
в”Ӯ
в””в”Җв”Җ No Consecutive
    в”ңв”Җв”Җ teacher_assign[(t, i)] + teacher_assign[(t, i+1)] вүӨ 1
    в””в”Җв”Җ subject_assign[(s, i)] + subject_assign[(s, i+1)] вүӨ 1
```
